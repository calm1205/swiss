# Day 1（9/5 土）: 旅の始まり、東京から空へ

> 東京 → 上海（経由）→ 機内泊

## フライト

### 成田 → 上海浦東
| 項目 | 詳細 |
|------|------|
| 便名 | MU524（中国東方航空） |
| 出発 | 13:50 NRT（成田） |
| 到着 | 16:15 PVG（上海浦東） |
| 所要時間 | 3時間25分 |
| クラス | エコノミー |

### 上海浦東 トランジット（9時間40分）
- 国際線トランジットエリア内で過ごす
- ラウンジ（有料 or クレジットカード付帯で利用可）
- レストラン・カフェあり
- 深夜便に備えて仮眠を取る
- **注意**: 中国の144時間トランジットビザ免除で入国も可能だが、深夜便のため空港内待機が現実的

### 上海浦東 → チューリッヒ
| 項目 | 詳細 |
|------|------|
| 便名 | MU255（中国東方航空） |
| 出発 | 01:55 PVG（9/6 日） |
| 到着 | 07:55 ZRH（9/6 日） |
| 所要時間 | 12時間 |
| クラス | エコノミー |

## 出発前チェックリスト
- パスポート（残存有効期限6ヶ月以上）
- eチケット控え
- 海外旅行保険の加入確認
- スイスフランの準備（空港ATMでも可）
- eSIM or SIMカードの事前手配

## 機内の過ごし方
- 上海→チューリッヒ（12時間）は深夜便。なるべく睡眠を取る
- スイス到着は現地朝8時 = 日本時間15時。到着後すぐ行動するため、機内で軽く朝食を摂る
- ネックピロー・アイマスク・耳栓を手荷物に
- モバイルバッテリーは上海の乗継時間中にフル充電しておく

## Tips
- 成田空港には11:30頃（出発2時間前）に到着推奨
- 上海浦東のトランジットエリアは広い。Terminal 1のラウンジや仮眠スペースを活用
- 機内食は2回提供される見込み（離陸後＋到着前）
- 時差: 日本→スイスは−7時間（サマータイム中）
