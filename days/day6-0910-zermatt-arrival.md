# Day 6（9/10 木）: 夕焼けに染まるマッターホルンとの出会い

> ブリエンツ湖 → ツェルマット【山岳エリア泊】

湖畔の余韻を残しつつ、マッターホルンの麓ツェルマットへ移動。午後はスネガ展望台へ。

## タイムライン（目安）

| 時間 | 内容 |
|------|------|
| 朝 | 湖畔で最後の散歩、朝食 |
| 9:30 | イゼルトヴァルト出発 |
| 12:30 | ツェルマット到着 |
| 13:00 | チェックイン、村でランチ |
| 14:30 | スネガ展望台へ（気が向いたら） |
| 15:00 | ライゼーでマッターホルン鑑賞 |
| 16:30 | ツェルマットに戻り、村を散策 |
| 18:00 | マッターホルンの夕焼け鑑賞 |
| 19:00 | ラクレットディナー |

## 見どころ

### 車窓の絶景（移動中）
- **ブリエンツ湖**: 出発時に船またはバスから湖を堪能
- **トゥーン湖**: インターラーケン〜シュピーツ間。エメラルドグリーンの湖面
- **シュピーツ城**: 湖に突き出た半島に建つ中世の城。車窓左手
- **ローヌ谷**: フィスプ手前の広大な谷。ブドウ畑が斜面に広がる

### ツェルマット村

![ツェルマット旧市街（ヒンタードルフ通り）](https://upload.wikimedia.org/wikipedia/commons/thumb/3/3a/4099_-_Zermatt_-_Hinterdorfstrasse.JPG/1280px-4099_-_Zermatt_-_Hinterdorfstrasse.JPG)

- **車両禁止の村**: ガソリン車は進入禁止。電気自動車とホースカートのみ
- **旧市街（Hinterdorf）**: 築数百年のヴァレー式木造家屋
  - ネズミ返しの石板（Stadel）が特徴的
  - 木造家屋越しにマッターホルンを撮れる構図
- 到着日は村をのんびり歩くだけで十分

### スネガ展望台（Sunnegga / 標高2,288m）

![ライゼーに映る逆さマッターホルン](https://upload.wikimedia.org/wikipedia/commons/thumb/6/66/Matterhorn_and_its_reflection_in_the_Leisee_October_2020.jpg/1280px-Matterhorn_and_its_reflection_in_the_Leisee_October_2020.jpg)

- ツェルマット駅からケーブルカーでわずか3分
- マッターホルンを最も美しく眺められるスポットの一つ
- **ライゼー（Leisee）**: 展望台近くの小さな山上湖
  - 風がない日は湖面にマッターホルンが映る
  - 湖の周りにベンチあり。ここでも読書やぼんやりできる
  - 秋はカラマツの黄葉とマッターホルンのコントラスト
- **無理に行かなくてもOK**。移動疲れがあれば村の散策だけで十分

### 夕焼けのマッターホルン

![夕焼けに染まるマッターホルン](https://upload.wikimedia.org/wikipedia/commons/thumb/c/c4/043_Matterhorn_reflecting_in_Stellisee_at_sunset_Photo_by_Giles_Laurent.jpg/1280px-043_Matterhorn_reflecting_in_Stellisee_at_sunset_Photo_by_Giles_Laurent.jpg)

- 日没時、山頂がオレンジ〜ピンクに染まる（アーベントロート）
- **おすすめ鑑賞スポット**:
  - キルヒブリュッケ（教会橋）周辺
  - ホテル・モンテローザ前の通り
- 9月上旬の日没: 19:30〜19:45頃

## グルメ

### ラクレット（Raclette）
- チーズの断面を熱で溶かし、茹でたジャガイモやピクルスにかける
- **Walliserkanne**: ツェルマットの定番フォンデュ＆ラクレット店
- **Restaurant Schäferstube**: ヴァレー州産チーズを使用
- 予算: CHF 30〜45/人

### カジュアルランチ
- **Café du Pont**: 橋のたもと、テラス席からマッターホルン
- **Bayard Metzgerei**: 地元の肉屋・デリ。トロッケンフライシュ（干し肉）のテイクアウト

## 移動詳細

### イゼルトヴァルト → ツェルマット
- イゼルトヴァルト → インターラーケン・オスト（バス 約20分 or 船）
- インターラーケン・オスト → シュピーツ（電車 約20分）
- シュピーツ → フィスプ（電車 約50分）
- フィスプ → ツェルマット（MGB鉄道 約65分）
- **合計**: 約3時間（乗換3回）
- スイストラベルパスですべてカバー

## 宿泊
- **エリア**: ツェルマット駅〜村中心部。マッターホルンが見える部屋を指定すると◎
- **Airbnb**: ◎（アパートメント・シャレー多数。キッチン付きで自炊も可能）
- **連泊推奨**: Day 6〜7の2泊

## Tips
- 移動日なので無理に予定を詰めない。到着後の散策＋夕焼け鑑賞で十分充実
- マッターホルンは午前が順光。本格的な撮影は翌日のゴルナーグラートで
- CoopとDennerが村内にあり、自炊する場合はここで食材調達

## 雨天時の代替プラン
- マッターホルン博物館（Zermatlantis）: 1865年の初登頂の歴史。約1.5時間
- 村のレストラン巡り。ヴァレー州産ワイン（Fendant, Dôle）のテイスティング
- 翌日のゴルナーグラートに備えてゆっくり休息
