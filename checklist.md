# チェックリスト

## 秋（9〜10月）のポイント
- **紅葉**: 山岳エリアは9月下旬〜10月中旬が見頃
- **天候**: 日中15〜20℃、山頂は0℃前後。レイヤー服装必須
- **日照時間**: 夏より短いが十分（6:50〜19:40頃、9月上旬）
- **メリット**: 夏のハイシーズンより混雑少、ホテルも比較的リーズナブル
- **ハイキング**: 秋は空気が澄み、山の眺望が最も美しい季節

## 持ち物
- [ ] 防寒着（山頂用のダウンジャケット）
- [ ] レイヤー対応の服装
- [ ] ハイキングシューズ（必須 — 本格的なトレイルあり）
- [ ] トレッキングポール（あると便利）
- [ ] サングラス・日焼け止め（高地の紫外線対策）
- [ ] 変換プラグ（Cタイプ）
- [ ] SBB Mobileアプリ（スイス鉄道の時刻表・チケット）
- [ ] 水筒（ハイキング用）

- [ ] 航空券（9/5チューリッヒ着・9/13ジュネーブ発）
- [ ] リギ山上ホテル（部屋数が限られるため早めに）
- [ ] ユングフラウヨッホ（オンライン事前購入で割引あり）
- [ ] シルトホルン（オンライン事前購入推奨）
- [ ] ゴルナーグラート鉄道
- [ ] マッターホルン・グレイシャー・パラダイス
- [ ] ホテル（秋でも人気エリアは早めに）
