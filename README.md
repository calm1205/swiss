# スイス旅行計画（秋・9日間）

- **時期**: 9〜10月
- **日数**: 9日間（移動込み）
- **人数**: 2人
- **テーマ**: 絶景・自然、山岳ハイキング、鉄道旅、グルメ
- **方針**: 到着日・出発前夜は都心部泊、それ以外は山岳エリアに滞在
- **ペース**: 予定を詰め込まず、ゆとりを持って過ごす。各日のタイムラインはあくまで参考
- **宿泊**: Airbnb活用（一部ホテルのみのエリアあり）

## ルート概要

チューリッヒ IN → リギ山 → ラウターブルンネン → ユングフラウ → ツェルマット → ジュネーブ OUT
