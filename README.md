# スイス旅行計画（秋・9日間）

- **時期**: 2026年9月5日（土）〜 9月13日（日）
- **日数**: 9日間（移動込み）
- **人数**: 2人
- **テーマ**: 絶景・自然、山岳ハイキング、鉄道旅、グルメ
- **航空便**: 中国東方航空・上海経由（往路: 成田→チューリッヒ / 復路: チューリッヒ→羽田）
- **方針**: チューリッヒ空港発着。宿泊費が高いチューリッヒ周辺は避け、コスパの良いエリアに宿泊
- **ペース**: 予定を詰め込まず、ゆとりを持って過ごす。各日のタイムラインはあくまで参考
- **宿泊**: Airbnb活用（一部ホテルのみのエリアあり）

## ルート概要

チューリッヒ空港 IN → ルツェルン → リギ山 → ラウターブルンネン → ユングフラウ → ブリエンツ湖 → ツェルマット → ベルン → チューリッヒ空港 OUT
