# 外的リスクと対策

体験の満足度を下げうる外的要因の一覧。事前に把握し、対策・代替プランを用意しておく。

## 天候

### 雨天・曇天
- **影響**: 展望台からの眺望ゼロ。ハイキングの魅力が大幅減
- **特に痛い日**: Day 4（ユングフラウヨッホ）、Day 7（ゴルナーグラート）
- **対策**:
  - 各日のファイルに雨天時の代替プランを記載済み
  - Day 4とDay 5は入れ替え可能（天気を見て判断）
  - 山頂のウェブカメラで当日朝に確認（jungfrau.ch, gornergrat.ch）
  - 雲の上に出られる場合もある — 麓が曇りでも諦めない

### 気温の急変・寒波
- **影響**: 9月下旬〜10月は山頂で降雪の可能性。ハイキングコース閉鎖
- **特に痛い日**: Day 5（ブリエンツ湖畔は問題なし）、Day 7（高地ハイキング）
- **対策**:
  - 防寒着のレイヤリングを徹底（checklist.md参照）
  - MeteoSwiss（スイス気象局アプリ）で週間予報を毎日チェック
  - コース閉鎖情報はSAC（スイスアルパインクラブ）のサイトで確認

### フェーン現象
- **影響**: アルプス北側で突然の高温・強風。頭痛を引き起こすことも
- **対策**: 水分補給、無理のないスケジュール

## 交通・インフラ

### 登山鉄道・ロープウェイの運休
- **原因**: 強風、雷、年次メンテナンス
- **影響**: 展望台へのアクセス不可。その日の計画が丸ごと崩れる
- **特に痛い日**: Day 4（ユングフラウ鉄道）、Day 7（ゴルナーグラート鉄道）
- **対策**:
  - 当日朝に運行状況を確認（SBB Mobileアプリ、各鉄道の公式サイト）
  - 日程に余裕を持たせているので、翌日にリトライ可能

### 季節閉鎖（候補Dの場合）
- **影響**: 10月に入ると一部の山岳施設・レストラン・ハイキングコースが閉鎖
- **対象**: シルトホルン、一部のリフト、山上レストラン
- **対策**: 9月中の日程（候補B）を選べば回避可能。候補Dの場合は各施設の営業期間を事前確認

### 鉄道の遅延・運休
- **原因**: 工事、事故、ストライキ（稀）
- **影響**: 乗り継ぎに失敗、到着遅れ
- **対策**:
  - SBB Mobileアプリでリアルタイム運行情報を確認
  - 乗り継ぎは余裕を持つ（最低15分）
  - スイスの鉄道は正確性が高いため深刻なリスクは低い

## 健康

### 高山病
- **影響**: 頭痛、吐き気、めまい、倦怠感
- **リスクのある日**:
  - Day 4: ユングフラウヨッホ（3,454m）
  - Day 7: マッターホルン・グレイシャー・パラダイス（3,883m）
- **対策**:
  - 急に走らない、ゆっくり行動
  - 水分を多めに摂取
  - 症状が出たら標高を下げる（1駅分降りるだけで改善することが多い）
  - 前日の飲酒を控える

### 時差ボケ
- **影響**: Day 1〜2で眠気・集中力低下。せっかくの景色を楽しめない
- **日本との時差**: −7時間（サマータイム中）
- **対策**:
  - Day 1はチューリッヒ市内の軽い散策にとどめている（対策済み）
  - 機内で現地時間に合わせて睡眠
  - 到着日は早めに就寝

### 日焼け・紫外線
- **影響**: 高地は紫外線が強烈。雪面の反射でさらに増幅
- **リスクのある日**: Day 4（ユングフラウヨッホ）、Day 7（ゴルナーグラート）
- **対策**: 日焼け止め（SPF50+）、サングラス必須

### 靴擦れ・足の疲労
- **影響**: ハイキングが苦痛に。後半の日程に響く
- **対策**:
  - ハイキングシューズは旅行前に履き慣らしておく
  - 靴擦れ防止パッド（コンピード等）を持参
  - Day 5（湖畔のんびりDay）が回復日として機能

## 混雑

### 観光地の混雑
- **影響**: ユングフラウヨッホ等の人気スポットで行列、写真に人が入る、レストランが満席
- **原因**: 団体ツアーバス（特に昼前後）、SW期間中の日本人旅行者
- **対策**:
  - 早朝出発（8:00台）で団体客より先に到着
  - 候補B・Dを選べばSW混雑を回避
  - レストランは事前予約

### 宿泊先の確保困難
- **影響**: 希望のエリア・価格帯で宿が取れない
- **特にリスクの高い宿**: リギ山上ホテル（部屋数が少ない）、イゼルトヴァルト（物件少）
- **対策**: 日程が決まり次第すぐに予約。Airbnbも早めに検索

## その他

### 為替変動
- **影響**: スイスフラン高で現地費用が増大
- **対策**:
  - 予算に幅を持たせて計画（budget.md参照）
  - 事前にある程度両替 or クレジットカード（為替手数料が低いもの）を用意

### 荷物の紛失・遅延（航空会社）
- **影響**: 防寒着やハイキングシューズが届かない → 山岳アクティビティに支障
- **対策**:
  - 防寒着・ハイキングシューズは機内持ち込みの手荷物に入れる
  - 最低限の着替えも手荷物に

### スリ・置き引き
- **影響**: チューリッヒ・ジュネーブ等の都市部で発生。パスポート・財布の紛失
- **対策**:
  - 貴重品は体に密着するポーチで管理
  - パスポートのコピーをスマホに保存
  - 混雑する駅・電車内で注意

### スマホ・通信トラブル
- **影響**: SBBアプリが使えない、地図が見られない、宿泊先と連絡取れない
- **対策**:
  - eSIM or 現地SIMを到着時に購入
  - 主要なルート・時刻表のスクリーンショットをオフラインで保存
  - SBBアプリのオフライン時刻表機能を活用
